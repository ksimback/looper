## Run Now In Session

That keeps design and execution in the same conversation.

## Run Outside The Session

Use the Python runner when you want to run later, repeatably, or outside the
LLM session:

```bash
python3 run-loop.py
```

The runner invokes only the argv arrays compiled into `loop.resolved.json`.
Review any cross-vendor council member before consenting to the first send.
It keeps lightweight state and logs, but it is not a durable orchestrator; use
one if you need scheduled runs, step-level retry, concurrency control, or
restart-safe production execution.
